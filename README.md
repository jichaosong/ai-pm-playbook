## 📖 概述

**AI PM Playbook** 是一份面向 AI 时代产品经理的实战知识库，涵盖：

- 🧠 **AI 产品框架** — AI SaaS、Agent、RAG、Human-in-the-Loop 产品的分析框架
- 📝 **PRD 模板** — 各类 AI 产品的产品需求文档模板
- 📊 **案例研究** — 真实 AI 产品从 0 到 1 的拆解
- 📈 **评估体系** — LLM 输出质量、Agent 成功率、产品级指标
- 🏗️ **Agent 产品设计** — 循环、记忆、权限、多 Agent、可观测性
- 🎯 **职业发展** — AI PM 成长路线图、技能图谱、作品集指南
- ✨ **提示词工程** — 写 PRD、用户研究、路线图的专属 Prompts

## 🎯 核心理念

**AI 产品 ≠ 套壳 API**

真正优秀的 AI 产品，需要：

1. **理解模型能力边界** — 知道 LLM 能做什么、不能做什么、什么时候不该用
2. **设计人机协作流程** — 不是全自动，而是在对的地方让 AI 做擅长的事
3. **建立评估闭环** — 没有评估就没有迭代，AI 产品的评估比传统产品复杂得多
4. **兼顾用户体验与可控性** — 用户既要自由度又要确定性，这是最大的设计挑战
